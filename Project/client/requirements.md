## Packages
framer-motion | Smooth layout transitions and feedback animations
lucide-react | Icons for the UI (already in base, but listing for clarity)
clsx | Conditional class names
tailwind-merge | Merging tailwind classes
react-hook-form | Form state management
zod | Schema validation
@hookform/resolvers | Zod resolver for react-hook-form
jspdf | Exporting resources to PDF
html2canvas | Helper for PDF export

## Notes
- Bilingual support (English/Arabic) requires `dir="rtl"` handling in layouts.
- Autism-friendly design focus: soft colors, clear fonts.
- API endpoints are defined in shared/routes.ts.
- Auth is handled via Replit Auth (/api/auth/user).
